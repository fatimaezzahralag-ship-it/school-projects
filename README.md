API de Gestion des Emprunts - Bibliothèque
Bienvenue dans le dépôt du projet de gestion des emprunts. Ce projet consiste en une API REST complète développée avec Laravel dans le cadre du module Technologies Backend
 Structure du Dépôt Pour
 une navigation simplifiée, le projet est organisé comme suit :LaravelTp2/ : Contient l'intégralité du code source de l'application Laravel (Modèles, Migrations, Contrôleurs et Routes).postman/ : Contient les captures d'écran de l'ensemble des tests effectués sur les endpoints, prouvant le bon fonctionnement de l'API et des codes de statut HTTP.
