📁 LaravelTp2/

Contient l’intégralité du code source de l’application Laravel, notamment :

les Modèles (Models)

les Migrations

les Contrôleurs

les Routes API

📁 postman/

Contient les captures d'écran des tests réalisés avec Postman.

Ces captures montrent :

les requêtes envoyées aux endpoints

les réponses de l’API

les codes de statut HTTP retournés par le serveur

Cela permet de prouver le bon fonctionnement de l’API.
